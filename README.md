# DocwTicToc
The DoCW Tic Toc Website

### Instructions
  - Install [node](https://nodejs.org/en/download/)
  - Clone the repo and install dependencies
  
        git clone https://github.com/UtkarshMe/DocwTicToc
        cd DocwTicToc
        npm install

  - Start the server
  
        node server.js
